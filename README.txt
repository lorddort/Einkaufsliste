##Einkaufsliste
Einkaufsliste ist ein Java Programm, welche zur Anfertigung einer Einkaufsliste dient.

##Benutzung
Man gebe in das Eingabefeld das Produkt ein, welches man später einkaufen will.
Durch die Schaltfläche "Hinzufügen" kann man das Produkt zur Einkaufsliste hinzufügen. Sobald man in der Einkaufsliste ein Produkt steht, kann man die Einkaufsliste speichern, entleeren, als PDF Datei erstellen, alte Einkaufsliste laden.
Der Speicherort ist frei auswählbar, so kann man die Datei wiederfinden und entsprechenden Optionen an der erstellten Datei durchführen kann.
Man beachte, dass man die erstellte PDF-Datei nicht mittels des Programms speichern kann.
